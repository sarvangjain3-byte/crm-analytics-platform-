# 🚀 CRM Analytics platform

A modern CRM and Business Analytics platform built using **React**, **Firebase**, and **Vite**.

---

## 📊 Features

- Secure Firebase Authentication
- Customer Relationship Management (CRM)
- Lead Management
- Sales Dashboard
- Analytics Dashboard
- Reports & CSV Export
- Follow-up Calendar
- Sales Pipeline
- AI Scanner (Work in Progress)

---

## 🛠 Tech Stack

- React
- JavaScript
- Firebase Authentication
- Cloud Firestore
- Vite
- Lucide React
